# gf_mm2twips

- **Module**: `_system` (Systeme)
- **Signature**: `decimal gf_mm2twips(integer ai_mmvalue, string as_orientation)`
- **Description**: Convertit des millimetres en twips

## Parametres

| Param | Type | Description |
|-------|------|-------------|
| `ai_mmvalue` | `integer` | Entier - mmvalue |
| `as_orientation` | `string` | Chaine - orientation |
