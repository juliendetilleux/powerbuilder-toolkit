# gf_time_from_minutes

- **Module**: `_masters` (Donnees de base)
- **Signature**: `time gf_time_from_minutes(long al_minutes)`
- **Description**: Convertit des minutes en temps

## Parametres

| Param | Type | Description |
|-------|------|-------------|
| `al_minutes` | `long` | Valeur long - minutes |
