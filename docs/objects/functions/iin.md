# iin

- **Module**: `_system` (Systeme)
- **Signature**: `boolean iin(string as_val, string as_arr[])`
- **Description**: Verifie si une valeur est dans un ensemble (In operator)

## Parametres

| Param | Type | Description |
|-------|------|-------------|
| `as_val` | `string` | Chaine - val |
| `as_arr` | `string` | Chaine - arr |
