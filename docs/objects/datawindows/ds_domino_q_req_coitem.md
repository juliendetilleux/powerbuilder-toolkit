# ds_domino_q_req_coitem

- **Type**: DataWindow
- **Style**: Grid
- **Module**: _manufg
- **Table principale**: 0

## SQL
```sql
  SELECT items.*,mfghead.mhreqqty,mfgcoitem.*
  FROM items join	mfghead on items.itcode = mfghead.mhitem left outer join mfgcoitem on  mfgcoitem.mccode=mfghead.mhcode
  where items.itcode in (:list_itcode)  and 
			mfghead.mhcode in (:list_mhcode)
			 
			
```
